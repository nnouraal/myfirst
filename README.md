# simple 

A simple and elegant SwiftUI app that celebrates Saudi identity and pride.

## 📱 About the App

This app displays a patriotic card featuring:
- The Saudi Arabian flag
- A proud statement "I am Saudi"
- 5-star rating with 1M+ reviews (placeholder)
- Heart and shield icons representing love and protection

## 🛠️ Technologies Used

- SwiftUI
- Xcode
- iOS / macOS development

## 📸 Features

- Rounded corner design with shadow effect
- Responsive image scaling
- Clean and modern UI layout using `ZStack` and `VStack`
- Custom styling with `.clipShape` and `.background`

## 🚀 How to Run

1. Clone this repository
2. Open `test.xcodeproj` in Xcode
3. Run the app on simulator or real device
